# Koi Tutorial

Welcome to Koi! If you’re new here, we recommend checking out the [introduction](/introduction) and [key concepts](/key-concepts) to learn more about
why we built Koi and the critical challenges it addresses in climate impact forecasting.
Koi combines cutting-edge methodologies and technologies to empower confident decision-making for the decarbonized economy.

In this tutorial, we’ll guide you through creating **composable, auditable, and evergreen** impact forecasts.
While this example might seem straightforward, it’s designed to highlight the core features and workflows of Koi.

In practice, forecasts can scale to analyze complex technologies and markets, but starting with a clear, relatable example will help you understand the fundamentals.
